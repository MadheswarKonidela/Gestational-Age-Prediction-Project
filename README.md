# Gestational Age Prediction using Machine Learning

## Project Overview

This project builds a machine learning pipeline to predict **gestational age** from clinical or medical data. Accurate gestational age estimation is important in prenatal care because it helps monitor fetal development and supports better medical decisions during pregnancy.

The notebook demonstrates the full workflow used in many real-world ML projects: data analysis, preprocessing, model training, and evaluation.

## How to Run the Project

1. Clone the repository

```
git clone https://github.com/MadheswarKonidela/Gestational-Age-Prediction-Project.git
```

2. Move into the project folder

```
cd Gestational-Age-Prediction-Project
```

3. Open the notebook

```
jupyter notebook gestational_age_prediction/GA.ipynb
```
